# 2026-06-20 작업 정리 · 마감 기록

> 견적 검토 화면(중앙 패널·좌측 메뉴·우측 위젯) 전반을 "단순·간결·하이라이트·직관" 컨셉으로 정리하고, 공종 상세 디자인을 전 공종에 데이터 연동으로 일괄 반영함.

## 1. 핵심 컨셉 (사용자 의도)
- 설명문 = **핵심 내용·키워드만**, 결과값 = **시각화 지표**(그래프·막대·눈금·아이콘).
- **직관 우선** — 보면 즉시 공감. 복잡·중복 회피.
- **단정 표현 금지**(거품·과다청구·부풀려진 단가). ZEROS = **최적합 견적 산출·제공 서비스**(시공 아님).
- **중장년 타깃** — 불필요한 영어 절제. 도형(박스) 남발 금지.
- 작업 원칙 전문: `AGENTS.md`(모든 AI 도구 공통).

## 2. 금일 변경 (확정)
### 중앙 패널 — 공종 상세 (전 공종 공통)
- **견적 작업 FLOW**: 공종 상세에서 제거 → 견적 검토 랜딩 히어로 박스로 이동(1곳).
- **상단 "주요 견적" 키워드 밴드** 신설 — 큰 키워드(Pretendard 800), 공종별 데이터.
- **히어로**: 좌=검토 항목 + "최대 N% 견적 차이" / 우=ZEROS Agent AI **견적 검증** + "최적합 견적 확정". 구버전(정직 단가 교정 모니터·도넛 게이지·거품 절감) **삭제**.
- **결과 박스**: "결과 지표 & Benefit" → **검토 후 제공 리포트**. 중복 수치·거품 문구·긴 유의점 제거, flat화.
- 영어 배지(PREPARE·PROCEDURE·PROVEN) 삭제, "정확한 진단을 위해…" 설명문 삭제.

### 견적 검토 랜딩 (renderReviewDesktop)
- 인트로 히어로 박스: 내용 삭제·박스 유지 → **견적 작업 FLOW** 배치.
- **HOW WE REVIEW**(STEP 1~3) 섹션 **삭제**(FLOW와 중복).
- 쇼케이스 카드: "평균 거품 절감" → **"평균 견적 최적화"**.

### 좌측 메뉴 (LeftSidebar)
- 공종·외주제작 메뉴의 **보충설명 문구 전부 삭제**(제목만). 견적규모별 금액 범위는 유지.

### 우측 위젯 (RightSidebar)
- 군더더기 라벨 삭제(선택한 검토 대상·실시간 검토 배지·신뢰도 부제·견적 예상가·중앙값 대비·신뢰 근거 제목).
- 슬라이더 → **범위 막대**(중앙값 눈금 + 최소/중앙값/최대). 신뢰 근거 3문장 → **아이콘 칩 3개**. CTA 설명문 삭제.
- **표시 토글**: 견적 검토 진입 시 숨김 → 우측 상단 "검토" 엣지 탭(음영 없음)으로 펼침, 헤더 `›`로 접기.
