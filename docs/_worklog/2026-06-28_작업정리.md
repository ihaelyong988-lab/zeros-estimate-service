# 2026-06-28 작업정리

## §1. 고객 화면 색 위계 청색화 + 공종 밴드 통일 (PHASE O-3)

사용자 캡처 주석 3장 기준. 메인화면 "AI 분석"과 동일한 청색(`#155EEF`)으로 강조색을 통일하고, 공종 상세 밴드를 견적총액별 방안에도 확장.

### 변경 내역 (항목별)
1. **사업소개 히어로 오렌지 → 청색 `#155EEF`** (`app/page.tsx` renderBusinessTab)
   - 헤드라인 "데이터로" (`text-accent` → `text-[#155EEF]`)
   - "안심 예산 대역 ±5%" — 텍스트·세로막대(`bg-accent`)·밑줄(`bg-accent`) 3요소 모두 청색
   - 푸터 "최적합 견적 검증" 라벨 청색
   - **CTA "무료 견적 검토 신청" 버튼은 오렌지 유지** (행동 유도 1곳만 강조 — AGENTS.md §10)
2. **공종 상세 밴드 라벨 `WORK_ITEM` → `주요 공종`** (`app/page.tsx` renderManualDetail, 약 1534행)
   - 색은 기존 청색 `#155EEF` 유지. 공용 컴포넌트라 전 공종 페이지에 자동 반영.
   - 영어 내부 토큰 → 한글 라벨(AGENTS.md §4 영어 절제 부합).
3. **견적총액별 방안에 동일 밴드 추가** (`app/page.tsx` TRADE_KEYWORDS, 약 91~103행)
   - `small`(온라인 간편검토)=소규모 분기·펌프·밸브 교체·단가 검토
   - `medium`(출장견적)=현장 실측·중규모 배관·계통 검토
   - `large`(프로젝트 사전진단)=대규모 설비·CAPEX·공법·예산
   - `renderManualDetail(selectedBudget)`가 small/medium/large로 호출되므로 키워드만 추가하면 동일 밴드가 자동 생성(밋밋함 해소).

### 의미축(색) 규칙 — 유지 필수
- **파랑 `#155EEF`** = 데이터·AI·검증 / **오렌지** = 행동(CTA)·주의(최대 N% 견적 차이)·비용(수수료 2%) / **네이비** = 본문.
