- **데이터 소스는 Supabase**(Mock 아님). `supabaseBrowser.ts` DEFAULT 키 + `.env.local` → `isSupabaseEnabled=true` 항상. `mock-data.ts` 시드는 안 보인다 → `SupabaseZerosService`에 멱등 시드해야 함.
- **배포 후 "똑같다" = 거의 PWA/브라우저 캐시.** 헤드리스 프리뷰는 스플래시(rAF)에서 멈춰 스크린샷 불가 → 검증은 `npm run build`+`show_widget` 시안. 사용자는 **시크릿 창**으로 확인.
- **마감처리 git은 단계별 출력 확인.** 한 줄 다중명령이 깨져 push가 안 나간 적 있음 → `xxx..yyy master -> master` 눈으로 확인.
