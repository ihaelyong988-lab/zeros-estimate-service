- ② 영어·이중설명 제거 → 한글 동사 키워드 단일화(자료 받기/자동 판독/거품 탐지/예산 산출/전문가 확인/검토서 발행).

### 2) 견적문의 자료등록 단계 탭 폼 — `components/forms/RequestWizard.tsx`
- ⑥ X표시 요소 제거: 헤더 "자료등록" 제목·"등록 방법을 선택하세요" 부제·하단 "접수하신…" 문구 삭제. 채널 카드=아이콘+제목+설명+하단 버튼형으로 정리.

- ⑧ 무료 견적 신청(quick) 4탭: ①사업·현장 → ②견적·연락(예상금액·전화·이메일) → ③참조사항 → 등록완료 탭.

- 완료 화면을 위저드 내부 탭으로 처리하도록 부모 `app/page.tsx` `onComplete` 라우팅 제거(미사용 `router`/`useRouter` 정리).
