# 2026-07-01 작업정리

## PHASE O-11 — AI Native 검증 SOP "내림차순 깔때기 인포그래픽" 최적화

### 배경 (의도)
- 사용자 캡쳐 ②의 손주석: "전체 내용을 인포그래픽 내림차순으로 직관 시각화 / 한글·영어 이중설명 중 하나만 / 폰트크기·하이라이트로 리듬감 / 설명문 함축 / 주제·절차·결과지표 재정립(중학생 눈높이) / 불필요한 아이콘·이중설명 삭제".
- 직전 PHASE O-10에서 영어·엔진·다크콘솔은 이미 제거됐으나, **funnel 폭 수렴이 97→85%로 거의 안 보이고(=그냥 6줄 리스트) 폰트 크기가 균일(18px)해 리듬감이 없었음**. 캡쳐는 품질 기준선이라 판단해 시각 임팩트를 재설계.

### 한 일
1. **진짜 내림차순 깔때기**: 폭 수렴을 또렷하게 — 모바일 완만(100→82%)·데스크톱 강하게(100→60%). `funnelW` 배열을 `w-[..] md:w-[..]` 페어 클래스로 교체하고, 컨테이너를 `items-center`로 중앙 정렬해 깔때기 형상이 눈에 들어오게.
2. **폰트·하이라이트 리듬감**: 단계별 `size` 필드 도입 — 핵심 단계(거품 탐지·검토서 발행)만 22px로 키우고 나머지 18px. 하이라이트는 거품탐지=accent(오렌지), 검토서=navy 강조 2곳만.
3. **설명문 함축(중학생 눈높이)**: 보조설명을 문장형 → **6~8자 키워드**로 압축("도면을 읽어 자재·물량을 뽑습니다" → "자재·물량 추출"). 결과칩(중복 요소) 제거해 한 줄에 [순번·키워드·함축] 만 남김.
4. **주제·결과 정리**: 헤드라인 "AI가 6단계로 걸러, 거품 없는 적정 예산까지" / 흐름 한 줄 "도면 한 장 → 자동 분석 → 전문가 확인 → 검토서". 결과지표 4종(예산밴드·리스크·범위·근거추적) 유지.
5. **접근성**: 깔때기 순번 색을 `text-gray-light` → `text-gray`로 상향(대비 4.5:1). 이모지 0·아이콘 0(타이포·폭·색만으로 위계).

### 검증
- `ui-ux-pro-max --design-system`: Funnel(3-step)·Trust & Authority·navy+blue CTA·이모지 금지 확인(기존 ZEROS 토큰과 일치).
- UI 품질 게이트 `--check`/`--pass`: 차단룰 R1 0건 통과(R2는 기존 KPI 라벨 경고, 비차단).
- `npm run build`: 0에러. `show_widget` 시안으로 깔때기 형상·리듬 확인.
