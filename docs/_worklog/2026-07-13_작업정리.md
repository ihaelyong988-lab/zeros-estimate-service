# 2026-07-13 작업정리

## PHASE O-30 — 사업 소개 탭 사업모델 중심 재구성 + ui-ux-pro-max 격상

### 배경 / 지시
- 사용자 캡쳐 지시: 사업 소개는 "웹사이트의 얼굴". 사업 목적·수단을 폰트·컬러로 **간결·단순·직관**하게 표현하고, 헤드라인 이하~최하단까지 **시각적 이동선**을 갖춘 디자인을 요구. 다른 페이지와 프레임이 상이하지 않게.
- 캡쳐 원문 4대 요지: ①목적=공사 지식 부족한 사업주에게 안심 견적 ②무료(소액 예산=자동화 툴 실적기반 무료)/유료(초과분=소정 수수료 신뢰 예산 책정) 2단 모델 ③현재=AI DB·실적 부족으로 인력 컨설팅 단계 ④최종 목표=AI Native 무료출장견적 → 고객 맞춤형 의사결정서비스.
- 기존 `renderBusinessTab`은 문제→분석→안심 3단계 타임라인만 담고 있어 사업모델(②③④)이 화면에 부재 → 단순 미화가 아닌 재구성 필요.

### 방향 확정 (AskUserQuestion 3건)
1. 구조 = **사업모델 중심 재구성**(3단계 타임라인 폐지 → 목적→무료/유료 2단→지금·목표 축→CTA).
2. 요금 = **무료·소정 수수료 2단 명시**(캡쳐 원문 그대로 정직 안내).
3. 헤드라인 = **현행 유지**("흔들리는 견적을 데이터로 최적의 예산을 책정합니다").

### 반영 (`app/page.tsx` `renderBusinessTab`)
- **히어로**: 헤드라인 현행 유지 + 목적 서브카피 1줄 신설(16px, 안심 견적 목적).
- **이렇게 제공합니다(2단)**: 무료(소액)=블루칩 `#E6F1FB/#0C447C` / 소정 수수료(초과분)=스틸 중립칩. 카드에 편집형 인덱스(01/02) + 호버 피드백(`transition-colors 200ms`·`motion-reduce` 가드).
- **여기서, 여기로(성장 축)**: 지금(요청사항별 전문가 컨설팅·AI DB 축적) → 목표(AI Native 무료출장견적·의사결정 서비스, ±5% 정확도 목표). 모바일 세로 스택(화살표 rotate-90).
- **신뢰 지표 클러스터(social proof) → CTA**: ui-ux-pro-max 권고(Social Proof-Focused, 패턴 Hero+Testimonials+CTA)대로 하단 한 줄 텍스트(30년·246건·98.4%)를 **24px tabular 대형 지표 3종 + 세로 헤어라인**으로 승격하고 사회적 증거를 CTA 직전에 배치. 검증 건수 246건만 블루 강조. 오렌지 CTA 단독.

### 디자인 규율 준수
- 오렌지 = 최종 CTA 1곳만(섹션 틱은 블루/스틸) · 이모지 아이콘 0 · 전 지표 `tabular-nums` · `motion-reduce` 존중 · 좌측 단일 축 이동선(헤드라인→제공→성장축→지표→CTA) · 스킬 제네릭 골드(#CA8A04) 미채택하고 확정 팔레트 유지.
